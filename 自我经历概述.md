# 自我经历概述

在我踏上计算机编程这条路之前，我是个学物理的，很长一段时间我都这么认为自己。

物理学理论让人着迷，而我这点智商恰恰够用，个别地方反应慢一点，整体上好好想想这些理论所要描述的东西，我还是觉得物理更有意思，但是后边接着选计算机绝对是物欲战胜了精神上的富足。

原子和分子怎么着，跟 0 和 1 怎么跳动，研究这些算不上，用的是这些东西，去实现目标，转化技术。

好吧。

其实我一开始的心思都已经偏了，我选择计算机是因为对虚无世界想象力的丧失，对现实世界想象力的勃发。

2022年8月23日，我的爸妈为他的宝贝儿子考上一个普通大学的应用物理系而感到欣慰，我说我想买台自己的笔记本电脑用于学习编程，老妈就在那天上午带我去了那家四年级上学期来我家上门装机的那个老板的电脑店，我选了当年新出的光影精灵 8，搭载着i5-12500h的cpu和3050显卡，我现在也还在使用这台主机去写下这些文字，因为至今为止，也过去不到四年，我马上就要毕业了。

但是说实话，我在高中往前，并不知道计算机到底在干什么，没什么概念，也没想过入这一行。

我买来电脑学习编程的初心，也是为了能够更好地去做科学计算或者建模什么的，在当时很长一段时间，我就是这么认为的，我甚至写下了一个名为 physicistscard 项目，一开始只是想要做个卡牌游戏，后来想做成零售商店，再后来最终成型的是一个复杂的网站，主要是用于科研工作者进行协作，当然我把零售商店也放进去了。

我目前没发现这个项目有什么起色，我在这个项目上投注了大量的业余精力，结果却少有所得，甚至到后边有点走火入魔，把学物理的时间也搭进去了，也就是在大二下学期那段时间，我几乎放弃了学习所谓的专业课，专注于去做这个项目，没成什么气候，只是一味的在堆砌代码。

时间回到2022年8月23日的那个下午，我踉踉跄跄地跟着b站上某个不知名的up主的操作，算是下载了一个3.7版本的 python ide，在shell中，打出了第一行 print("Hello,world!")，想念那个时候的知足感，仅仅是在shell中输入命令得到回应就已经感到无比自足了。

理工男，呵呵。说实话我真的很讨厌别人拿这个标签来定义我，专业确实是理学范畴，做的工作也是工学范畴，但是我本人却并没有那种理工男刻板印象的特征，相反，我就是喜欢能给别人带来印象不错的路人感，就不错了。

我喜欢大刘所描述的科幻世界，东野圭吾笔下的推理视角，前者让人脱身于琐事之外，投身于更宏伟的世界去理解与感受，后者让人在琐事之内，也会发现世事洞明皆学问，其实就是侦探的视角或者说对于人性的描述。好吧，在遇到这些感兴趣的东西之前，我从来没有发现过自己这些面，还有些人，遇到之后，也是。

喜欢的东西，就慢慢靠近，讨厌的东西，就直接远离。

太符合人性了。可我有个哥们偏偏说我这是为了更大的欲望，又或者说，人类的欲望是大他者的欲望，我们所做的一切事情都是没什么意义的。因为本质上不是自己真正想做的？看有趣的书，跟有趣的人约会，吃美味的事物，欣赏有趣的节目。有的时候，我都不知道该怎么接话了，我记得有段时间我的头脑可犀利了，似乎天地万物皆可入我胸怀，我的口中亦能随意吞吐天下之物。其实，说实话，就是以为自己开悟的状态，我仍在大他者的欲望中。

我很确定，在我长大的过程中，我对那么几个人是十分感兴趣的。可我发现，越是喜欢的，反而留不在身边。

是爱？是好感？还是感兴趣？理由可多了。

不过我发现在一开始是有互相吸引的情况存在的，不过我真的不爱装，至少在我错过这些人之前，我都不知道他们是装好的。

别人表现什么，我就回应什么，我回应什么，别人就接着回应什么。

这是我一贯的交互模式，我承认确实有些心思复杂的人，也知道他们是什么样子，如果他们演出了什么样子，那就相信什么样子吧。

我上了大学才第一次体会跟有过一定交集的人不再交谈是一件挺让人难以接受的事情，这种感受放大之后，就会造成所谓的社交性麻木，即不再关心自己身边的人，只关心能够产出回应的关系，回应就是目的，人类在某种程度上也是种抽象的计算机模型。
