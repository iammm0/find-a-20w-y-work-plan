**Secbot 智能化自动化渗透测试机器人**
2026.01-至今

**技术选型：** Python, LangChain, Playwright, SQLite-Vec, TypeScript, React Native

**项目描述：** 一个AI驱动的自动化安全测试平台（仅授权安全测试）。采用LangChain框架集成大模型能力，赋予系统智能化的漏洞检测与渗透测试决策能力。使用Playwright进行浏览器自动化测试，SQLite-Vec支持向量化数据存储与相似度检索，通过TypeScript与React Native构建跨平台用户交互界面，实现Web端与移动端的统一操作体验。

**责任描述：**

1. 基于LangChain集成大模型能力，实现AI驱动的漏洞分析与渗透测试流程编排，支持自动化安全扫描决策。
2. 利用Playwright框架自动化Web应用安全测试，支持复杂交互场景的漏洞发现与验证。
3. 采用SQLite-Vec实现漏洞知识向量化存储与语义检索，支持历史漏洞关联分析与智能推荐。
4. 设计跨平台用户界面，采用React Native与TypeScript实现移动端与Web端的一体化交互体验。
5. 发布至PyPI官方包库，支持pip便捷安装与集成，促进社区使用与反馈。
