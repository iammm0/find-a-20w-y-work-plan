**waibi-waibi MBTI性格工坊**
**2025.10-至今**

**技术选型：** TypeScript, Next.js, React, Tailwind CSS

**项目描述：** 基于MBTI人格分类的交互式聊天与训练平台。通过人格化AI助手为用户提供个性化的对话体验和学习训练服务。平台包含人格测试与分析、AI聊天互动、个性化训练课程等核心功能模块；前端采用Next.js + React构建，使用Tailwind CSS进行样式设计，提供流畅的用户界面和交互体验。

**责任描述：**

1. 设计和实现MBTI人格相关页面与组件，包括人格卡片、人格实例管理、人格聊天交互界面。
2. 开发用户认证、个人资料、头像编辑等账户管理功能模块。
3. 实现聊天功能的UI组件，包括聊天窗口、侧边栏、消息展示等，支持实时对话交互。
4. 设计并优化训练课程相关页面，包括训练预览、训练状态展示、进度跟踪。
5. 构建邀请系统、订阅购买、用户支持等辅助功能的前端界面。
6. 优化全局样式与布局结构，完善layout与provider配置，提升UI一致性与用户体验。
