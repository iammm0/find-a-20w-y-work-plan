# 春招求职准备 / Work Plan

个人春招求职计划与资料仓库，用于跟踪投递、复习与面试准备。

## 仓库结构

| 路径 | 说明 |
|------|------|
| **day1/** ~ **day4/** | 每日准备笔记（春招准备第 N 天） |
| **books/** | 参考资料 PDF（如《你好算法》《智能体设计模式》等） |
| **resumes/** | 求职岗位简历 PDF（AI Agent 研发、Golang 后端等） |
| **experiences/** | 经历与实习简历（按公司/项目分目录，Resume.md） |
| **projects/** | 个人项目简历（按项目分目录，Resume.md） |
| **me/** | 一寸职业照等个人资源 |
| 根目录 | 计划表、春招投递名单、复习资料链接、面试笔记、自我经历概述等 Markdown |

## 复习与准备方向

- 书籍：《深入剖析 Kubernetes》《Go 程序设计语言》《Python 程序设计语言》《智能体设计模式》等
- 刷题：牛客网笔试、LeetCode 算法（如热题 100 道中等题）
- 工具：简历中技术栈基础 + Cursor / OpenCode 等 AI 编程工具使用

## 使用说明

本仓库为个人工作区，按需更新计划文档与简历版本，并通过 Git 提交记录进度。
