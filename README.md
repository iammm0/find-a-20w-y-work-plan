## 仓库结构

| 目录 / 文件 | 说明 |

| **day1/**、**day2/**、**day3/** | 每日准备笔记（原 plan1 已迁至 day1） |
| **resumes/** | 求职岗位简历 PDF（AI Agent 研发工程师、Golang 后端工程师等） |

| **projects/** | 个人项目简历与描述（PhysicistsCard、WaibiWaibi、SensorAI、SecBot、ComsolAgent 等） |
| **experiences/** | 公司/实验室经历（如科创中心、搜狐等） |
| **计划表.md** | 近期待办与复习计划 |
| **春招投递名单.md** | 春招投递企业名单（Golang 后端 / AI Agent 研发梯队） |
| **复习资料链接.md** | 复习资料链接（Go 面试、LeetCode 热题 100 等） |
| **面试笔记.md**、**自我经历概述.md** | 面试笔记与自我概述 |
